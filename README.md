Capstone to Udacity's ML Engineering Nanodegree

## Data

Data is available to [download from Kaggle](https://www.kaggle.com/c/kkbox-music-recommendation-challenge/data). The notebooks use the files:
- members.csv
- songs.csv
- train.csv
- test.csv
